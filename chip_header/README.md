## Include Paths

Ensure the following include paths are added in STM32CubeIDE:

* ${ProjDirPath}/chip_header/CMSIS/Include
* ${ProjDirPath}/chip_header/CMSIS/Device/ST/STM32F4xx/Include
* ${ProjDirPath}/chip_header/DSP/Include

### How to configure

1. Right-click on the project → **Properties** (or press Alt + Enter)
2. Navigate to **C/C++ General → Paths and Symbols**
3. Go to the **Includes** tab (GNU C)
4. Click **Add** and select the paths above
